Module Suite.Sync

module Sync: sig .. end
type test_fun = unit -> unit
A synchronous test function.
val bracket : (unit -> 'a) -> ('a -> unit) -> ('a -> unit) -> test_fun
bracket setup test teardown generates a Sync.test_fun which will use setup to create state needed for the test, then pass that state to test, and finally will pass that state to teardown.
